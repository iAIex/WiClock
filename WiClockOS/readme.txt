Before starting, you have to download Python 2.7 (32bit) and Pygame with the latest release (32bit)!

To test the v1 of WiClockOS, start widgets.py
To try out custom widgets, you have to edit in widgets the test.ini.
To add custom fonts, add a file ended with .font in the font folder, look into the default5x3.font
for for the syntax.

Importand:
- So far modules are not supported!
- many bugs, that not a stable version
- just one widget supported
- the hardware interface to raspberry not included so far


Tested under Win 10
